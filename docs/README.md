# Documentation

Ce dossier contient toute la documentation du projet organisée par catégorie :

## Structure

- **`architecture/`** - Documentation technique et guides d'architecture
  - `ARCHITECTURE_GUIDE.md` - Guide d'architecture du système
  - `DECISION_GUIDE.md` - Guide des décisions techniques
  - `IMPLEMENTATION_SUMMARY.md` - Résumé de l'implémentation
  - `API_DOCUMENTATION_FOR_UI.md` - Documentation API pour l'interface

- **`tasks/`** - Résumés d'implémentation des tâches
  - `TASK_*_IMPLEMENTATION_SUMMARY.md` - Résumés détaillés des tâches 4-14

- **`ui/`** - Documentation interface utilisateur
  - `UI_DESIGN_PROMPTS.md` - Prompts de design UI
  - `UI_QUICK_START_GUIDE.md` - Guide de démarrage rapide UI
  - `KIRO_UI_PROMPT.md` - Prompts spécifiques à Kiro
  - `UI_AGENT_CONFIG.json` - Configuration agent UI
  - `TYPESCRIPT_TYPES.ts` - Types TypeScript

## Navigation

Consultez le README principal à la racine pour les instructions d'installation et d'utilisation.
