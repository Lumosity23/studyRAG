Cours d'Électronique - ESP32

Introduction

L'ESP32 est un microcontrôleur développé par Espressif Systems. Il intègre WiFi et Bluetooth dans un seul chip.

Caractéristiques techniques

  • Processeur: Dual-core Xtensa LX6
  • Fréquence: jusqu'à 240 MHz
  • RAM: 520 KB SRAM
  • Flash: 4 MB (minimum)
  • WiFi: 802.11 b/g/n
  • Bluetooth: v4.2 BR/EDR et BLE

Applications

L'ESP32 est utilisé dans de nombreuses applications IoT:

  • Capteurs connectés
  • Systèmes domotiques
  • Objets connectés
  • Prototypage rapide

Programmation

L'ESP32 peut être programmé avec plusieurs environnements:

  • Arduino IDE
  • ESP-IDF (framework officiel)
  • MicroPython
  • PlatformIO

Exemple de code

#include "WiFi.h"

void setup() {
  Serial.begin(115200);
  WiFi.begin("SSID", "password");
  
  while (WiFi.status() != WL_CONNECTED) {
    delay(1000);
    Serial.println("Connecting to WiFi...");
  }
  
  Serial.println("Connected to WiFi");
}

void loop() {
  // Code principal
}

Conclusion

L'ESP32 est une solution puissante et polyvalente pour les projets IoT. Sa facilité d'utilisation et ses nombreuses fonctionnalités en font un choix populaire pour les développeurs.
